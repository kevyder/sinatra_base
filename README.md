# Sinatra Base

A base for your projects on Sinatra

## Usage

Clone this repository executing:

```bash
git clone https://github.com/kevyder/sinatra_base.git
```

And execute:

```bash
cd sinatra_base
```

To install the project gems execute:.

```bash
bundle install
```

To run the project execute:

```bash
rackup -p <PORT>
```

## Install pre-commit

```bash
pre-commit install
```
